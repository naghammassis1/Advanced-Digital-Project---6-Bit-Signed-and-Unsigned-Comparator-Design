🔢 6-Bit Signed & Unsigned Comparator

📌 Overview
This project implements a 6-bit digital comparator that compares two numbers and determines whether they are greater than, less than, or equal. It supports both signed and unsigned modes.

⚙️ Features
Compare two 6-bit inputs
Supports:
Signed (2’s complement)
Unsigned numbers
Outputs:
A > B
A < B
A == B
Two implementations:
Structural (logic gates)
Behavioral (Verilog operators)

🧪 Testing
A testbench is used to verify the design by comparing both implementations and ensuring correct results for all inputs.

🛠️ Technologies
Verilog HDL
Digital Logic Design
Testbench Simulation
